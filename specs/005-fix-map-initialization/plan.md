# Implementation Plan: 지도 초기화 및 레이아웃 수정

## Summary

로그인 후 메인 페이지 진입 시 발생하는 레이아웃 오류(사이드바 중앙 배치)를 수정하고, 브라우저 Geolocation API를 사용하여 지도의 초기 중심을 사용자의 현재 위치로 설정합니다. 위치 권한 거부 시에는 강남역을 기본 위치로 사용합니다.

## Technical Context

**Language/Version**: TypeScript 5.x, Node.js 20.x  
**Primary Dependencies**: React 18.3.1, react-kakao-maps-sdk 1.1.27, Firebase SDK 10.13.0, Tailwind CSS 3.4.10  
**Storage**: Firebase Firestore (NoSQL)  
**Testing**: Vitest (Unit), Playwright (E2E)  
**Target Platform**: Web (Desktop/Mobile Responsive)  
**Project Type**: Web Application  
**Performance Goals**: Search filter < 0.5s, Smooth map transition  
**Constraints**: Kakao Map API key security, Firebase Security Rules
